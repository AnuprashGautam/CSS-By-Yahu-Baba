Welcome to the Home Page

This is an example of a simple responsive layout. Try resizing the browser window to see the navigation bar automatically change to a sidebar when the width of the container is reduced. This is made possible by the CSS Container Query, a powerful new feature in CSS.

The container query allows us to apply styles based on the size of a parent element (the container) rather than the entire viewport. This provides more granular control over responsive designs, making components truly self-contained and adaptable to any layout.
